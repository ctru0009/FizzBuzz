## Getting Started

### Prerequisites

- Node.js (v18 or later)
- .NET 8.0 SDK
- Postgresql
- Docker

### Running the Application

1. Clone the repository:
```bash
git clone https://github.com/ctru0009/FizzBuzz
cd FizzBuzz
```
2. Run app with docker
```bash
docker-compose up --build
```
3. Open URL and enjoy!
```
localhost:3000
```
